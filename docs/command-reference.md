# 🎯 Command Reference

This document lists all available commands in the Claude Productivity Suite.

## Agent-OS Commands

### `/plan-product`
Plans a new product or feature with structured approach.
- Creates project roadmap
- Identifies technical requirements
- Suggests architecture
- Estimates timeline

### `/create-spec`
Creates detailed specifications for features.
- User stories
- Technical requirements
- API design
- Database schema
- UI/UX considerations

### `/execute-tasks`
Executes development tasks based on specifications.
- Implements features
- Creates tests
- Handles errors
- Commits code

### `/analyze-product`
Analyzes existing codebase for improvements.
- Architecture review
- Performance analysis
- Security assessment
- Upgrade recommendations

## Codebase-OS Commands

### `/analyze-codebase`
Comprehensive code health analysis.
- Complexity metrics
- Duplication detection
- Dependency analysis
- Performance bottlenecks
- Security vulnerabilities

### `/clean-codebase`
Automated code cleanup.
- Remove dead code
- Organize imports
- Fix formatting
- Update dependencies
- Optimize assets

### `/refactor-smart`
Intelligent refactoring suggestions.
- Split large components
- Extract common code
- Improve patterns
- Enhance performance
- Add type safety

### `/monitor-quality`
Monitor code quality trends.
- Track metrics over time
- Alert on degradation
- Generate reports
- Suggest improvements

## Testing Commands

### `/fix-and-test`
Complete error fixing and testing workflow.
- Static analysis
- Unit testing
- Integration testing
- E2E testing
- Performance testing

### `/pre-deploy-check`
Quick 5-minute deployment safety check.
- Build verification
- Security scan
- Console cleanup
- Basic smoke tests

### `/test-core-flows`
Test essential user journeys.
- Authentication
- Core features
- Data persistence
- Mobile responsiveness

### `/railway-deploy`
Railway-specific deployment validation.
- Environment setup
- Health checks
- Migration readiness
- Performance optimization

## Workflow Commands

### `/daily-dev`
Complete daily development workflow.
- Morning: Code health check
- Development: Feature building
- Testing: Quality assurance
- Evening: Progress review

### `/pre-deploy`
Pre-deployment checklist workflow.
- Code quality check
- Test execution
- Security validation
- Documentation update

### `/prod-release`
Production release workflow.
- Final testing
- Deployment preparation
- Monitoring setup
- Rollback planning

### `/emergency-fix`
Emergency deployment process.
- Critical checks only
- Fast validation
- Immediate deployment
- Post-deployment monitoring
